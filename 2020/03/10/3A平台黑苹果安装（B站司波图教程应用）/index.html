3A平台黑苹果安装（B站司波图教程应用）

Posted on 2020-03-10 |

注：所有资料来源B站司波图：（https://www.bilibili.com/video/av94747725?from=search&seid=1517022325829447483）

前期资料

  1. U盘2个
  2. 资料下载（https://pan.baidu.com/s/1FBq20_ZRipJkQ2Kb6CvRlQ）
  3. 硬件兼容性查询（msi X470 gaming plus + ryzen2600 + RX580 成功）

U盘制作

  1. 打开gibMacOS.bat

  2. R 下载恢复镜像

  3. 5 （full install 15.3）

  4. 回车 Q 退出

  5. Make install

  6. 选择U盘

  7. yes 删除分区

  8. 找到下载的PKG文件，复制路径到cmd 写入镜像

  9. 输入4I（替换默认的Clover）最新无法启动

  10. 看到新硬盘。删除新硬盘中EFI，用教程中底包2替换。

  11. 进入EFI/OC查看驱动（具体看视频10:00）

注入驱动（修改config，有现成可跳过）

  1. 模板 x其他/0.5.5/doc/sample.plst复制到3.制作基础config（已有需改好的，可以直接用）

  2. 复制到U盘/efi/oc

  3. 重命名为config.plist

  4. 3config/propertree file/open/u盘/config

  5. ctrl+shift+r(ctrl+r基本一致)选择efi/oc

  6. snapshot创建快照

  7. 修改ssdt的config。其中kernal/patch重要

  8. 先删掉patch，从3/amdb补丁/patch里的kernal/patch复制到config里面。

  9. Quirks里面的几个电源管理选项打开

  10. misc（杂项）watchdog

  11. security/allow /scan->0

  12. nvram/ui ->01/02

  13. boot-args 启动设置 24:33 显卡设置

  14. prev-lang 语言设置

    PLATFOTM跳到后面

  15. quirks设置

    完成

  16. 加入硬件驱动 从4硬件驱动复制到oc/kexts

  17. 3.config/propertree/propertree 打开config ctrl+r efi/oc 注入

  18. kernal可以看到驱动

  19. 无线网卡驱动注意顺序

  20. agdpmod删掉（我用的RX580） npci=0x2000看above4gdcoding (31:00)

  21. PLATFOTM 3./gensmb 生成机型信息 iMacpro1,1 /Macpro7,1

  22. type复制到-systemproname serial-serialnumber board–mlb uuid–uuid

  23. 验证不过多试几次

  24. U盘完成

硬盘处理

  1. 新u盘安装winpe（链接在资料里有）
  2. 重新启动后在winpe里删除原本分区
  3. 设置为gpt格式
  4. 重启

bios处理

  1. 关闭快速启动
  2. 关闭CSM启动
  3. 关闭安全启动
  4. 找到peripherals/above4g 启动
  5. 找到hand-off 打开
  6. Ostype设置为 win8.1/10（这项没有找到，微星主板）

安装

  1. reset nvram
  2. macos 恢复
  3. 磁盘工具/抹掉/APFS
  4. 重新安装
  5. 第一次重启-installer-第二次安装
  6. 初始化

拷贝引导

  1. boot/efi
  2. 搜索terminal
  3. sudo diskutil list 找到目标硬盘efi文件夹
  4. sudo diskutil mount disk0s1 挂载efi文件夹
  5. 在访达里打开efi 把boot/efi 复制进去
  6. 拔掉U盘重启

完成

存在的问题：关闭CSM后
