3A平台黑苹果安装（B站司波图教程应用）

发表于 2020-03-10 Disqus：

准备工作

前期资料

  1. U盘一个

  2. 资料下载（https://pan.baidu.com/s/1FBq20_ZRipJkQ2Kb6CvRlQ）

  3. 硬件兼容性查询

U盘制作

  1. 打开gibMacOS.bat

  2. R 下载恢复镜像

  3. 5 （full install 15.3）

  4. 回车 Q 退出

  5. PKG

  6. Make install

  7. 选择U盘

  8. yes 删除分区

  9. 找到下载的PKG文件，复制路径到cmd 写入镜像

  10. 默认写入Clover

  11. 输入4I（替换Clover）最新无法启动

  12. 看到新硬盘。删除新硬盘中EFI，用教程中底包2替换。

  13. 进入EFI/OC查看驱动（具体看视频10:00）

    注入驱动（修改config，有现成可跳过）

    1. 模板 x其他/0.5.5/doc/sample.plst复制到3.制作基础config（已有）

    2. 复制到U盘/efi/oc

    3. 重命名为config.plist

    4. 3config/propertree file/open/u盘/config

    5. ctrl+shift+r(ctrl+r基本一致)选择efi/oc

    6. snapshot创建快照

    7. 修改ssdt的config。其中kernal/patch重要

    8. 先删掉patch，从3/amdb补丁/patch里的kernal/patch复制到config里面。

    9. Quirks里面的几个电源管理选项打开

    10. misc（杂项）watchdog

    11. security/allow /scan->0

    12. nvram/ui ->01/02

    13. boot-args 启动设置 24:33 显卡设置

    14. prev-lang 语言设置

      PLATFOTM跳到后面

    15. quirks设置

      完成

  14. 加入硬件驱动 从4硬件驱动复制到oc/kexts

  15. 3.config/propertree/propertree 打开config ctrl+r efi/oc 注入

  16. kernal可以看到驱动

  17. 无线网卡注意顺序

  18. agdpmod删掉 npci=0x2000看above4gdcoding (31:00)

  19. PLATFOTM yong 3./gensmb 生成机型信息 iMacpro1,1 /Macpro7,1

  20. type复制到-systemproname serial-serialnumber board–mlb uuid–uuid

  21. 验证不过多试几次

  22. U盘完成

硬盘处理

  1. winpe

  2. 删除分区

  3. 调整gpt

bios处理

  1. 关闭快速启动
  2. 关闭csm快速启动
  3. 关闭安全启动
  4. peripherals/above4g 启动
  5. hand-off 打开
  6. ostype win8.1/10

安装

  1. reset nvram
  2. macos 恢复
  3. 磁盘工具/抹掉/APFS
  4. 重新安装
  5. 第一次重启-installer-第二次安装
  6. 初始化

拷贝引导

  1. boot/efi
  2. 搜素-terminal
  3. sudo diskutil list
  4. 找到efi sudo diskutil mount disk0s1
  5. 打开 把boot/fei 复制进去
  6. 拔掉U盘重启

完成
