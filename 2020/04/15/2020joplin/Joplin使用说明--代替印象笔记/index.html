Joplin使用说明–代替印象笔记

1. 简介

Joplin是一个开放源代码的笔记和待办事项应用程序。您可以将笔记整理到笔记本中并进行标记。Joplin还提供了一个Web剪辑器，可以从互联网上保存文章。

如果你喜欢 印象笔记，您对开源软件不会感到太不舒服
Joplin是一款出色的开源笔记记录应用程序，具有很多功能，您可以通过将笔记记录与Dropbox和NextCloud等云服务链接来记录笔记，待办事项并在设备之间同步笔记。 。
Joplin还有一个Web Clipper，可让您将网页另存为笔记。 Web Clipper可用于Firefox和Chrome / Chromium浏览器。
Joplin允许导入Enex格式的Evernote文件，从而使从Evernote的切换更加容易。
由于您拥有数据，因此可以以乔普林格式或原始格式导出所有文件。

Markdown基础

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档

Markdown标题

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

Markdown段落和字体

Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车。

5. 网页剪藏

下载

下载剪藏插件
全选所有文件，点击操作–下载
保存到平时不用的文件夹（推荐新建文件夹 C:\Program Files\Google\Extension）
解压文件到当前文件夹

添加进chrome浏览器

在浏览器输入 chrome://extensions/
右侧打开开发者模式
加载已解压的扩展程序
找到刚才的文件夹C:\Program Files\Google\Extension选择文件夹
可以在右上角看到已添加的扩展程序，点击报错，2分钟后回复正常

使用

点击插件
选择想要保存的笔记本
选择保存网页的格式
