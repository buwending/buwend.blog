分享两篇文章，地铁上无聊看看就好

Posted on 2020-04-21 |

地铁上无聊的话看看就好

前言

这里面有些是大企业的套路，小企业的话无所谓的。完全不用紧张，就当坐地铁兜兜风好了…

第一篇

1.简单介绍下你自己

分析： 这似乎是必答题。不要以为这很容易，这一个环节可能分分钟让HR看出你这人怎么样！如果你用一分钟来重复你简历上的个人介绍，恭喜你，你的印象加分就没有了！你应该首先介绍下自己的姓名+学校+专业+学历+家乡，起码让HR对你有一个基本的认识吧！然后话峰一转，精简、准确、流畅的表达一下自己在大学最那得出手的实践经历或实习经历或项目成果或学习成果等，但切记你引用的素材（1到2个即可，按照你主要做了什么？结果是怎样的？你从中学到了什么？这样的黄金法则来叙述！）是与你所面试的职位所需要的条件是相关的！以此来突出自己的优势或强项。记住一定要在短时间内激发起HR对你的好感！

推荐模版： 您好，面试官，我叫XXX，是XX省XX市人，今年6月将从XX学校XX专业本科（专科）毕业。除了简历上您看到的个人介绍，我愿意在这里特别说一下我在XXX方面的特长/我最大的特点是XXX/我具备很好的XXX能力（然后给出相关事例）。正是基于自己对这方面的自信，使我有勇气来应聘贵公司的XXX这一职位。在这时候，可以装下逼看下表，然后说：“一分钟到了，希望我没有超时，以上就是我的一个简单自我介绍，谢谢！”同时附以很阳光的微笑！给HR留下好印象！

2. 你最大的优点和缺点？

分析： 从自己的优点说起，中间加一些小缺点，最后再把问题转回到优点上，突出优点的部分；优点可以答沉着冷静、条理清楚、乐于助人和关心他人、适应力和幽默感、乐观和友爱等。

3. 你有哪些兴趣爱好？

分析： 回答的兴趣爱好可以体现自己积极主动的性格、团队合作的精神、深入学习的意愿等。

4. 你朋友对你的评价？

分析： 想从侧面了解一下你的性格及与人相处的问题。

推荐模版： 我的朋友都说我是一个可以信赖的人。因为，我一旦答应别人的事情，就一定会做到。如果我做不到，我就不会轻易许诺。或者说我觉得我是一个比较随和的人，与不同的人都可以友好相处。在我与人相处时，我总是能站在别人的角度考虑问题。

5. 你做过哪件最骄傲的事或成功的事？

分析： 这是HR给你的一个机会，让你展示自己把握命运的能力，你就拿出自己曾经做值得讲的事说吧！最好选跟职位相关的经历，同时按照“你做了什么？结果是什么？从中你学到了什么？”这样的模式来讲，简洁，精确，流畅！

6. 你人生中最令你受挫的事情是什么？

分析： 这个问题是为了了解你对挫折的容忍度及调节方式。可以说让你受挫的经历，但更重要的是你如何处理、克服或改变。

7. 说说你的家庭？

推荐模板： 虽然我的父母都是普通人，但是从小我就看到父亲工作一丝不苟、认真负责，我母亲为人善良，特别乐于助人，父母的一言一行一直在教导我做人做事的道理。

8. 为什么你还没有找到工作？

分析： 没有找到工作并不是因为能力不行，而是在寻找更好的机会。可以说：“我正在谨慎选择我的工作，也有一些工作可以选择，但并不是非常满意。”

9. 你的实习或实践收获是什么？

分析： 为了考察你的胜任力，通常HR会问你在实习期间的收获。此时，不要夸大自己的成绩，谦虚一点。还有，不妨说一下自己的失误然后是怎么解决的（不用怕，毕竟你是在实习，以体现你的诚恳）。你应该首先精简、准确、流程的表达出你当时实习的主要工作内容或过程或你采取了哪些行动，实习的结果是怎样的（用具体的数字或成果体现出来，而不要用一些模糊的概率！）？然后学到或得到了什么？或者表达出你在那段实习经历中出现过什么失误（有就表达，没有不要刻意捏造！）并怎么解决的然后通过总结得到了什么经验教训？

10.你并非毕业于名牌学校？

分析： 不卑不亢地回应。比如，我认为是否毕业于名牌院校并不是最重要的，重要的是我目前掌握的技能完全可以胜任工作。而且我的经历让我比一些名牌院校毕业生的动手能力更强，我想我更适合这个职位。

11. 你认为你在学校属于好学生吗？

分析： 如果成绩好，可以说：“是的，但判断一个学生是不是好学生有很多标准，还包括思想道德、实践经验、团队精神、沟通能力等，我是一个全面发展的学生。”如果成绩不尽理想，可以说：“我认为好学生的标准是多元化的，我的学习成绩还可以，在其他方面我的表现也很突出，如我去很多地方实习过，我在学生会组织过许多活动，有比较强的团队合作精神和组织能力。”

12. 你在大学里的学习成绩不敢恭维，你如何解释呢？

分析： 这类问题往往是一种“压迫法”的面试技巧，提到的弱点不是问题关键所在，HR是为了看你在压力下如何反应，所以应沉着应付，想法扭转劣势，表现出自己的急才与修养。

13.请谈谈你的职业生涯规划是什么？

分析： 面试前，你要了解下你面试的职位的晋升渠道，起码有一个概念性的认识。一般是：助理——专员——资深专员——主管——资深主管——部门副经理——部门经理（当然也有其他的晋升渠道）。最保险的回答应该先说明你要发展或进取的专业方向，并表明你脚踏实地的工作态度。“我的职业规划是勇于进取，将我的全部精力与专业知识融入到我所从事的工作中去。首先我会选择进入到公司的某个职位，接着通过这个职位不断学习和锻炼自己，并通过努力争取给公司带来更大的利润价值，然后希望在今后几年中，成为一名内行的专业人士，很清楚地理解自己的行业、公司、及最大的挑战以及机会之所在。到那时，我想我未来的发展目标便会清晰地显露出来。”类似于这样的回答会使你远远地高于你的同龄人。

温馨提示: 在回答这个问题时，很多应届毕业生都会落入圈套，答道：“管理阶层”，自以为能表明自己的雄心壮志。这是一个老掉牙的答案，还很可能会引发HR一系列提问，而你会感到无法回答：管理阶层的定义是什么?一个经理的基本责任是什么?做什么领域的经理?面对这个问题时。其实你不需要回答得多么完美，你办不到；也不要说得太通俗，HR会认为你胸无大志；也不要回答太专业，HR可能会觉得你好高鹜远。

14. 谈谈如何适应新环境？

分析： ①遵守规章制度，不迟到不早退，不擅自离岗；②根据领导指示和工作安排，置顶工作计划，提前准备并按计划完成；③多请示并及时汇报，遇到不明白的要虚心请教；④多学习，努力提高自己的业务水平。

15.你会如何处理别人的批评？

推荐模板： 沉默是金，不必说什么，否则情况更糟，我会等大家冷静下来再讨论，也会接受建设性的批评。

16. 你对加班的看法？

分析： 实际上好多公司问这个问题，并不证明一定要加班，只是想测试你是否愿意为公司奉献。

推荐模板： 如果是工作需要我会义不容辞加班，我现在单身，没有任何家庭负担，可以全身心的投入工作。但同时，我也会提高工作效率，减少不必要的加班。

17. 你对跳槽的看法？

分析： ①正常的跳槽能促进人才合理流动，应该支持；②频繁的跳槽对单位和双方都不利，应该反对。

18.你对薪资的要求？

分析： 如果你对薪酬的要求太低，那显然贬低自己的能力；如果你对薪酬的要求太高，那又会显得你分量过重，公司受用不起。一些雇主通常都事先对求聘的职位定下开支预算，因而他们第一次提出的价钱往往是他们所能给予的最高价钱，他们问你只不过想证实一下这笔钱是否足以引起你对该工作的兴趣。

推荐模版： 我对工资没有硬性要求，我相信贵公司在处理我的问题上会友善合理。我注重的是找对工作机会，所以只要条件公平，我不会计较太多。或者可以说我参加过相关的职业培训，不需要进行大量的培训，因此，我希望公司能根据我的情况和市场标准的水平，给我合理的薪水。当然如果HR必须要你自己说出具体数目，请不要说一个宽泛的范围，那样你将只能得到最低限度的数字。最好给出一个具体的数字，这样表明你已经对当今的人才市场作了调查，知道像自己这样学历的雇员有什么样的价值。

19. 你为什么选择来我们公司工作？

分析： 最糟糕的回答就是“因为我很喜欢你们公司”，明显答非所问嘛！人家问的是你喜欢人家公司的具体理由，以体现你在面试之前对他们公司的了解和认识。在回答这个问题时，得提前对公司所处的行业、公司内部情况、职位情况等做出一些调查了解，然后就可以根据HR的需求来回答。

推荐模版： 1、从公司行业上谈你的见解。2、谈谈你对公司的了解。3、可以说说公司对新入职员工的培养及你有相对应的优势，另外你可以重点提下公司的突出优势或特点。

20. 你能为我们公司带来什么呢？

分析： ①假如你有培训经历的话，可以说“我在那里接受过多久的职业培训，可以很快适应上岗工作”以显示你可以给公司降低职业培训精力或者费用。②应再次重复自己的优势，你可以说：“就我的能力，我可以做一个优秀的员工在组织中发挥能力，给组织带来高效率和更多的收益”。企业喜欢求职者就申请的职位表明自己的能力，比如申请营销之类的职位，可以说：“我可以开发大量的新客户，同时，对老客户做更全面周到的服务，开发老客户的新需求和消费。”等等。

21. 为什么选择这个职务？谈谈你对这个职位的理解？

分析： 表明这一职务一直是自己的兴趣和专长，也累积了一定的经验及人脉，相信自己一定能胜任。可以适时决出过去取得的成绩，但避免过于夸张或过于炫耀。建议你面试之前多看看你将要面试的具体要求和职责即可，同时结合一些自己的理解去讲就ok。

22. 对这项工作，你有哪些可预见的困难？

分析： ①不宜直接说出具体的困难，否则可能令HR怀疑你不行。②可以尝试迂回战术，说出你对困难所持有的态度——工作中出现一些困难是正常的，也是难免的，但是只要有坚忍不拔的毅力、良好的合作精神以及事前周密而充分的准备，任何困难都是可以克服。一般问这个问题，面试者的希望就比较大了，因为已经在谈工作细节，但常规思路回答，又被面试官“骗”了。当面试官询问这个问题的时候，有两个目的。第一，看看应聘者是不是在行，说出的困难是不是在这个职位中一般都不可避免的问题。第二，是想看一下应聘者解决困难的手法对不对，及公司能否提供这样的资源。而不是想了解应聘者对困难的态度。

23.你缺乏工作经验，如何能胜任这项工作？

分析： 要体现出诚恳、机智、果敢及敬业。

推荐模板： 作为应届毕业生，在工作经验方面的确有所欠缺，因此在读书期间我一直利用各种机会在这个行业里做兼职和实习。实际工作远比书本复杂，但我有较强的责任心、适应能力和学习能力，而且很勤奋，所以是一个优秀的实习生，也积累了一些经验。请放心，学校所学及兼职的工作经验使我一定能胜任这个职位。

24.与上级意见不一时，你将怎么办？

分析： 一般可以这样回答“我会给上级以必要的解释和提醒，如果上级坚持己见，我会服从上级的意见。”另外需要注意的是：如果你回答“对于非原则性问题，我会服从上级的意见，对于涉及公司利益的重大问题，我希望能向更高层领导反映。”必死无疑。你没有摸清楚公司的内部情况，先想打小报告，这样的人没有人敢要。

25.你很努力，但客户始终不满意并投诉，你怎么做？

分析： ①首先，我会保持冷静，积极应对；②其次，我会反思原因，一是看自己在解决问题上是否有考虑不周之处，二是看客户是否不了解情况，三是看客户是否故意刁难；③再次，如果自己确有步骤，按照规定作出合理安排；如果是客户的要求不符合规定，我会明确指出；④把处理情况向领导汇报，不会因客户投诉而丧失工作热情。

26.工作中你难以和同事、上司相处，你该怎么办？

分析： ①我会服从领导的指挥，配合同事的工作。②我会从自身找原因，仔细分析是不是自己工作做得不好让领导不满意，同事看不惯。还要看看是不是为人处世方面做得不好，如果是这样的话我会努力改正。③如果我找不到原因，我会找机会跟他们沟通，请他们指出我的不足，有问题就及时改正。④作为优秀的员工，应该时刻以大局为重，即使在一段时间内，领导和同事对我不理解，我也会做好本职工作，虚心向他们学习，我相信，他们会看见我在努力，总有一天会对我微笑的。

27.如果你做的一项工作受到上级领导的表扬，但你主管领导却说是他做的，你该怎样？

分析： 我首先不会找那位上级领导说明这件事，我会主动找我的主管领导来沟通，因为沟通是解决人际关系的最好办法，但结果会有两种：①我的主管领导认识到自己的错误，我想我会视具体情况决定是否原谅他。②他更加变本加厉的来威胁我，那我会毫不犹豫地找我的上级领导反映此事，因为他这样做会造成负面影响，对今后的工作不利。

28.假设你在单位工作，成绩比较突出，得到领导的肯定。但同时你发现同事们越来越孤立你，你怎么看这个问题？你准备怎么办？

分析： ①成绩比较突出，得到领导的肯定是件好事情，以后更加努力。②检讨一下自己是不是对工作的热心度超过同事间交往的热心了，加强同事间的交往及共同的兴趣爱好。③工作中，不伤害别人的自尊心。④不在领导免前拨弄是非。

29.你希望与什么样的上级共事？

分析： 最好回避对上级具体的希望，多谈对自己的要求。

推荐模板： “作为刚步入社会的新人，我应该多要求自己尽快适应环境，而不应该对环境提要求，只要发挥能力，做好巩固走即可”。或“我希望我的上级能够在工作中对我多指导，对我工作中的错误能够立即指出。”

30.如果我们没有录取你，你下步有什么打算？

分析： 可以表示会认真总结这次面试的经验，找原因、找差距，但也不会因此气馁，会把这次的不成功当做历练积累。

31.如果我录用你，你将怎样开展工作？

分析： ①如果你对于应聘的职位缺乏足够的了解，最好不要直接说出自己开展工作的具体办法。②可以尝试采用迂回战术来回答，如“首先听取领导的指示和要求，然后就有关情况进行了解和熟悉，接下来制定一份近期的工作计划并报领导批准，最后根据计划开展工作。这个问题的主要目的也是了解应聘者的工作能力和计划性、条理性，而且重点想要知道细节。但思路中所讲的迂回战术，并不普适，如果面试官引导了几次仍然是回避的话，基本上不会录用你了。

32.如果你被录用了，但工作一段你发现你并不适合这个职位，你怎么办？

分析： 首先强调，申请这个职位是你经过了深思熟虑和全面了解的决定，所谓的不适应可能是业务不熟练导致的。而后表明态度，如果发现短时间内能力与职位有差距，你会不断学习，虚心向领导和同事请教经验，力争减少差距。

33.除了本公司外，还应聘了哪些公司？

分析： 就算不便说出公司名称，也应回答同质的公司，如果应聘的其他公司是不同业界，容易让HR觉得你求职意向不明确。

34.你何时可以入职？

分析： 最好回答“如果被录用的话，到职日可按公司要求的时间入职”。如果做不到，可以告知用人单位，但需要表明愿意在能力所及范围内尽快到岗。

35.你有什么问题要问我吗？

分析： 面试官的这个问题看上去可有可无，其实很关键，企业不喜欢说“没问题”的人，因为其很注重员工的个性和创新能力。企业不喜欢求职者问个人福利之类的问题，如果有人这样问：贵公司对新入公司的员工有没有什么培训项目，我可以参加吗？或者说贵公司的晋升机制是什么样的？企业将很欢迎，因为体现出你对学习的热情和对公司的忠诚度以及你的上进心。

第二篇

自我介绍不仅仅是自我介绍。答的好，是妥妥的加分项，答的不好，就是自己给自己挖坑。

要知道，HR会接着你的自我介绍，问很多的问题。

一个人，是否重视这次面试，是很能够体现在“自我介绍”这个环节的。后面的面试，也许你不好掌控，但自我介绍，完全是你一个人做主，是最能够提前做好准备的。

说实话，临场反应再差的求职者，都可以把自我介绍准备好。

如果你这个环节都说不好，你觉得，面试官又怎么会认为，你重视这次面试，真诚想要加入呢？

先说具体如何进行自我介绍：

3分钟左右。

请涉及：对面试公司的热爱及了解、过往重点工作经历（匹配求职意向的内容）、简单的未来规划。

要求更高的同学，建议准备2种。

1、前面讲的3分钟内容。

2、1分钟精简版，适用于群面等时间比较紧的场合。

给一个最近指导的例子，可能比较直观。

同学根据提示，自己写的

修改后的

除了前面说到的模板，内容方向需要注意的是：

1、结合招聘岗位，只讲重点。

简历内容这么多，实际的你，更有很多很多可以描述的东西。但时间有限，没有面试官会听你说个没完。

根据你求职的岗位，说重点即可。

其实简历制作的原则，也是一样。只是自我介绍时间更短，内容更精华。

2、有理有据，少说空话。

如果你说“自己学习能力强”，这就是一句假大空的话。谁都可以说自己学习能力强。

你如果真的在这方面有突出，就要举一个例子，比如是1个月从0到1考了什么证书等。

如果你说自己负责过视频的制作，就要说出来，具体负责了什么。

3、有开头有结尾，有逻辑。

开头问候，结尾总结。中间1、2、3条理清楚。

4、特别提醒。

自我介绍中的内容，很可能是面试官后续发问的内容。所以，一是要引起重视，讲最重要的，你最想让面试官知道的内容。 而是做好准备，扬长避短，不要给自己挖坑。

比如前面你说自己学习能力强，但是没有举例子。

面试官很可能，顺着你的话问你，怎么证明你学习能力强呢？ 结果你答不上来，或者是非常普通的成绩，根本不能算是“学习能力强”。 那你就是给自己挖坑了。

其实自我介绍也是每个人各有不同，说起来也只能提供大致的思路。和你具体的求职岗位、经历、个人风格有很大的关系。

当然，自我介绍只是面试的第一步。后续的回答也非常的重要。

下面是进阶版：我着重来谈谈，如果在自我介绍后，应对HR的问题。

（很厉害的内容，建议收藏，面试前再拿出来看看哈哈）

前面说到，自我介绍不是自我介绍本身，是面试的开场。

从这个不一样的角度，帮助大家理解自我介绍，应该说些什么。

自我介绍后，HR会问你很多问题。如果这些问题，你早有准备，就不会觉得慌乱，不知道说什么好了。

如何回答HR的问题？——万变不离其宗：运用案例进行讲解。

HR可能问到你各种奇奇怪怪的问题，但是仔细分析起来，其实也没有多少类具体的问题（根据不同岗位，面试重点不同）。

只是某一类问题，有多种不同的问法而已。

以“学习能力”举例。比如，HR问：
A、你的学习能力如何？
B、能否举一个例子，证明你在短时间内，完成了一件你以前并不会的事情？
C、在以往的工作过程中，你是否有遇到过很有挑战的事情。你是如何应对的？

如果用方式A问你，你一下子就明白了。 但是用B、C来问你，你就不知道说什么好了？
其实都是同一类问题（C可以有很多种回答方向，除了学习能力，你也可以回答抗压能力、团队合作能力等。）。
千万不要干巴巴的说，“我是一个学习能力很强的人”。这也太没有说服力了，谁都可以说自己有学习能力。一定要举出实际例子。
你都用同一个准备好的，证明你“学习能力”的案例，都能够回答。

这样听起来，是不是就简单了很多？

下面说说，如何准备面试所需的案例。

1、案例搜集：

面试前，建议大家认真梳理自己以往的经历，把各种自己觉得很棒的事例，都一一列举，写下来。

不然等到面试时，再来想，很容易脑子卡壳，一下子想不到特别好的。

一开始只需要把所有想到的，都写下来。

思考方式：想不出来的时候，可以根据岗位需求，通过4个问题来寻找素材，准备案例
——我有没有这样的经历？
——当时是怎么做到的？（或者为什么没有成功？）
——我为什么要这么做？
——是如何确保实现的？（或者给我带来了什么收获？）

3、案例归纳：

对案例进行分类，看看这个案例，可以用于证明自己哪类方面的特质。

注意，最后你整理出来的案例，数量不一定多，3-5个足够了。

（比如1个证明你学习能力强，1个证明团队协作强，1个证明逻辑思维缜密等。 根据不同的求职方向，准备的案例方向会有所不同）。

（如果你不清楚到底应该准备哪些方面的案例，欢迎私信我）

回答此类问题时，你举的案例一定具有以下特质：

1）真实（可以适当美化）。

你需要通过讲故事的方式，在面试时来阐述这个案例。所以真实是首要基础。并且真实的经历，才能避免HR追问细节时，你回答不上来。

2）恰当。

你需要识别时，HR为什么问你这个问题（我前面已经回答了）。

然后，从你准备好的案例中，选择一个恰当的。

同样是“学习能力”案例举例：
例如有的候选人面试技术岗位，说到学习能力，虽然也可以举滑冰的例子，但是滑冰的学习能力，主要体现在肢体协调性。和技术岗位的脑力学习能力，还是有比较大的差别的。
这种情况，最好是举技术方面的例子。
比如多少时间内，从0开始，你是如何学习了一种新的语言/算法（看书、查询网络、收集经典案例、咨询前辈、从简单程序开始尝试、不断试错并总结……），然后是如何完成了一个什么样的程序开发。（一般情况下，别的人需要多少时间才能完成，但是你只用了多少时间就完成了。）

3）回答时，注意逻辑和总结。

虽然案例是你早就准备好的，但是也不要像背课文一样。

听到HR提问后，可以稍微停顿几秒，表示你在思考。

然后用讲故事的方法，娓娓道来。

结尾时，可以加一些总结。

比如，我知道我自己在XX方面，可能经验不是非常足，但是我能通过学习，并且付出比其他人更多的努力，马上弥补上。

除了自我介绍本身，也欢迎大家通过模拟面试的方式，提前熟悉应聘岗位的提问重点。类似于考试押题，知道考点，也提前练习了解题思路，自然胸有成竹，能够有更好的发挥。
